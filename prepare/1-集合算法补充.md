# 集合算法

## STL 集合算法（`<algorithm>`）

前提：输入必须有序。`set`/`map` 天然有序，`vector` 需先 `sort`。

```cpp
// vector 输出 → back_inserter（推荐先 reserve 减少扩容）
vector<int> C; C.reserve(max_len);
set_intersection(A.begin(), A.end(), B.begin(), B.end(), back_inserter(C));

// set 输出 → inserter（set 没有 push_back，必须用 inserter）
set<int> C;
set_intersection(A.begin(), A.end(), B.begin(), B.end(), inserter(C, C.begin()));
```

四个算法统一形式，只差"何时输出"的语义：

```cpp
set_intersection        (a.begin(),a.end(), b.begin(),b.end(), out); // A ∩ B
set_union               (a.begin(),a.end(), b.begin(),b.end(), out); // A ∪ B
set_difference          (a.begin(),a.end(), b.begin(),b.end(), out); // A - B
set_symmetric_difference(a.begin(),a.end(), b.begin(),b.end(), out); // A △ B (属于A或B但不同属)
```

## 用 set 成员函数手动实现

不依赖 `<algorithm>` 时，遍历 + 成员函数即可：

```cpp
// 交：遍历 A，在 B 中查找
for (int x : A) if (B.count(x)) C.insert(x);
// 并：全部倒入
C = A; C.insert(B.begin(), B.end());
// 差：遍历 A，排除在 B 中的
for (int x : A) if (!B.count(x)) C.insert(x);
```

## 广义交 / 广义并（多集合）

```cpp
// 广义交：k 个集合的公共元素 —— 初始化为第一个，逐对归约
set<int> ans = sets[0];
for (int i = 1; i < k; ++i) {
    set<int> tmp;
    set_intersection(ans.begin(), ans.end(),
                     sets[i].begin(), sets[i].end(),
                     inserter(tmp, tmp.begin()));
    ans = move(tmp);
}
// 广义并：k 个集合的全部元素 —— 直接逐个 merge
set<int> ans;
for (auto& s : sets) ans.insert(s.begin(), s.end());
```
