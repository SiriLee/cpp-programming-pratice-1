# 基础算法速查

快速幂 / 质数判断 / 生成质数

## 快速幂（二分幂，O(log n)）

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

template<typename T>
T qpow(T a, ll n) {
    T res = 1;
    while (n) {
        if (n & 1) res = res * a;
        a = a * a;
        n >>= 1;
    }
    return res;
}
```

> **取模版**：乘法和结果加 `% MOD` — `res = res * a % MOD`，`a = a * a % MOD`。见 EOJ 1016: A^B mod C。

## 质数

```cpp
// 判断 n 是否为质数（6k±1 优化，O(√n)）
inline bool is_prime(ll n) {
    if (n < 2) return false;
    if (n == 2 || n == 3) return true;
    if (n % 2 == 0 || n % 3 == 0) return false;
    for (ll i = 5; i * i <= n; i += 6)
        if (n % i == 0 || n % (i + 2) == 0) return false;
    return true;
}
```

> **6k±1 优化**：所有 ≥5 的质数必可写成 6k±1 形式，因此每次只试除 i 和 i+2（即 6k-1 和 6k+1），跳过 2/3 的候选。
