# 题型模板补充

## 动态规划

### 线性 DP（计数型）

```cpp
// 台阶走法：每次可走 1~k 步，求方案数
dp[0] = 1;
for (int i = 1; i <= n; ++i)
    for (int step = 1; step <= k && step <= i; ++step)
        dp[i] += dp[i - step];                   // 计数 = 累加；最值 = max/min
```

> 若递推只依赖前 k 项，可用 k 个变量滚动，省去数组。

### LCS（最长公共子序列）

两串 s1、s2，`dp[i][j]` = 前缀对前缀的 LCS 长度。相等从左上角 +1，不等取左/上较大者。

```cpp
vector<vector<int>> dp(n + 1, vector<int>(m + 1));
for (int i = 1; i <= n; ++i)
    for (int j = 1; j <= m; ++j) {
        if (s1[i-1] == s2[j-1]) dp[i][j] = dp[i-1][j-1] + 1;
        else dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
    }
// 答案: dp[n][m]
```

> 编辑距离变形：不等时 `min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1`。

### LIS（最长上升子序列）

无序序列中找最长严格递增子序列（不要求连续）。

```cpp
// O(n²)：n≤1000 够用。dp[i] = 以 a[i] 结尾的 LIS 长
vector<int> dp(n, 1);
for (int i = 0; i < n; ++i)
    for (int j = 0; j < i; ++j)
        if (a[j] < a[i]) dp[i] = max(dp[i], dp[j] + 1);
// 答案: *max_element(dp.begin(), dp.end())

// O(n log n)：tail[k] = 长 k+1 的 IS 的最小末尾，二分维护
vector<int> tail;
for (int x : a) {
    auto it = lower_bound(tail.begin(), tail.end(), x);
    if (it == tail.end()) tail.push_back(x);
    else *it = x;
}
// 答案: tail.size()
```

> `lower_bound` = 严格上升；非严格（不下降）改用 `upper_bound`。

## 排序：非标准格式 → normalize → sort

Maya 历、不同进制……通用套路：**先解析为可比类型，再 `operator<`**。

```cpp
unordered_map<string, int> mp = {{"pop",0}, {"no",1}, ...};

struct Date {
    int d, m, y;                                 // 全部已是 int
    Date(string ds, string ms, int ys)
        : d(stoi(ds)), m(mp.at(ms)), y(ys) {}    // normalize 在此完成
    bool operator<(const Date& o) const {
        if (y != o.y) return y < o.y;
        if (m != o.m) return m < o.m;
        return d < o.d;
    }
};
```

> 1052（不同进制排序）同理：`stoll(s, nullptr, base)` 统一转十进制——只改 normalize 那一步。

## 括号匹配题型

### 题型 → 方法

### 多类型判合法（1020）

```cpp
bool check(const string& s) {
    stack<char> st;
    for (char c : s) {
        if (c == '(' || c == '[' || c == '{') st.push(c);
        else {
            if (st.empty()) return false;
            char t = st.top(); st.pop();
            if ((c == ')' && t != '(') ||
                (c == ']' && t != '[') ||
                (c == '}' && t != '{')) return false;
        }
    }
    return st.empty();
}
```

### 分数计算：哨兵 + 合并（1029）

栈中 `LEFT` 哨兵标记 `(` 位置，数值为子表达式分数。`push_val` 处理 `AB` 加法合并。遇 `)` 时：栈顶是 LEFT → `()` 贡献 1；栈顶是数值 → `(A)` 贡献 2×A。

```cpp
const int LEFT = INT_MAX;

stack<int> st;
auto push_val = [&](int val) {
    if (st.empty() || st.top() == LEFT) st.push(val);
    else { int prev = st.top(); st.pop(); st.push(prev + val); }
};

for (char c : s) {
    if (c == '(') st.push(LEFT);
    else {
        if (st.top() == LEFT) { st.pop(); push_val(1); }       // ()
        else { int v = st.top(); st.pop(); st.pop(); push_val(v * 2); } // (A)
    }
}
return st.top();
```

> 若计分规则换了（如 `()=0`、`(A)=A+1`），只改 push_val 合并逻辑和倍数，哨兵框架不变。

### 最长合法括号子串

栈存**下标**而非字符。栈底始终是"最后一个无法匹配的 `)` 的下标"作为左边界（初始 -1）。

```cpp
stack<int> st; st.push(-1);                      // 虚拟左边界
int ans = 0;
for (int i = 0; i < s.size(); ++i) {
    if (s[i] == '(') st.push(i);
    else {
        st.pop();
        if (st.empty()) st.push(i);              // 当前 ')' 多余 → 成为新边界
        else ans = max(ans, i - st.top());       // 匹配成功 → 计算长度
    }
}
```

> 例：`())()` → `()` 合法，长度 2。
